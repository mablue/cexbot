# cexbot
A cryptocurrency portfolio tracker with telegram support


## Installation

1) Download the repo:
`git clone https://github.com/mablue/cexbot.git`
  1.1) `cd cexbot`
  1.2) `pip3 install virtualenv`
  1.3) `virtualenv .env`
  1.4) `source .env/bin/activate`

3) Install all requirements:
`pip install -r requirements.txt`

3) Change Uppercase letters in the CEXBOT.py
`YOUR EXCHANGE API KEY`
`YOUR EXCHANGE API AECRET`
`YOUR TELEGRAM API BOT TOKEN`
`YOUR CHAT ID`

4) Run the bot:
`python CEXBOT.py`
